# project react-covid

project yang mengunakan library dari react js untuk membuat sebuah website tentang perkembangan dari virus corona di seluruh dunia.

## clone project

git clone

## start project

npm install -> npm start

## api corona virus

https://corona.lmao.ninja/v2/

## depedencies

1. bootstrap
2. react-svg
3. node-sass
4. react-map-gl
5. react-number-format
